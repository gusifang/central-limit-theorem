An interactive browser app illustrating two fundamental ideas in probability:

Sampling convergence: as the number of rolls increases, the observed histogram approaches the exact distribution.

Central Limit Theorem: as the number of independent dice increases, the distribution of their total approaches a normal distribution.

The app supports both fair and highly skewed dice, making it possible to explore the CLT beyond the usual fair six-sided die examples.

Live demo: https://gusifang.github.io/central-limit-theorem/
